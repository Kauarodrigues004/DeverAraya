Qual é o elemento HTML correto para definir um formulário?

Resposta: <'form'>

Resposta: <'form action='/action_page.php' target='_blank'><'/form>

Qual é o elemento HTML correto para agrupar dados relacionados em um formulário e também, por padrão, desenhar uma borda ao redor deles?

Resposta: <'fieldset'>
